# Fiction Faith Bible

A minimalist tarot card reference app with meanings, keywords, and interpretations.

## Features

- Browse Major Arcana and Minor Arcana (Wands, Cups, Swords, Pentacles)
- Search cards by name
- Card details: upright/reversed meanings, keywords, love/career/finances interpretations

## Setup

```bash
pnpm install
pnpm dev
```


## Tech

- Next.js 16, React 19, TypeScript
- Tailwind CSS, Framer Motion, Font Awesome, Lucide
